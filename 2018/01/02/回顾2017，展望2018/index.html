回顾 2017，展望 2018

发表于 2018-01-02 | 分类于 生活 | 浏览 次

时间过得好快，一会儿的功夫又是新的一年。

其实我这人不太喜欢写年终总结，一是自己比较随性，没有立 Flag 的习惯，所以也没啥总结可写；二是不想写，比较懒（主要还是懒，看吧，这篇总结本该是昨天写的）。

但是想着也不小了，年轻人应该给自己点压力，立个目标。正好去年也刚从学校毕业，总不能刚一毕业就把自己给「颓」了吧。

回顾 2017

回顾过去的一年，总体来说负面状态占据了比较多的时间。

年初由于毕业被延期了导致某一段时间内情绪比较低落，做啥事都没劲。后来瞒着导师偷偷去公司实习，每天担心受怕的，生怕导师突然叫你去找他，就这么诚惶诚恐得结束了自己的学生生涯。

毕业后接着就面临买房的压力。虽说刚毕业的学生不该考虑房子的事情，但是考虑到家里能帮衬点而且老爸也叫我平时留意下房子的事情，所以渐渐的关注起了楼市动态。

对于杭州这种热点城市，家里的那点帮衬在动辄六成甚至全款首付的杭州楼市前面显得微不足道。在「杭州即将跃升一线城市」的舆论环境中，想着若不早点买房，恐怕以后都上不了车。于是哪怕是心灰意冷也抱着一线希望在窥探着。

工作上，作为一名开发人员，对于分配的任务尽量以做好的态度去完成，但是在技术上却没有太多的积累。原本打算学习下前端的相关知识，后来也不了了之。

不过过去的一年也是有所收获的。

年初加入了「stormzhang」的小密圈，在圈内学习了很多以前都没接触过的知识，突然觉得自己以前 low 爆了。某种意义上说加入小密圈提升了我的认知，而当时第一时间付费加入也是过去一年中最正确的一笔投资。

第二个收获就是学会了理财。过去一年通过理财我也多了一笔额外的收益，虽然不是特别多，但是对于刚毕业的我来说足够满意了。后来的「中兴员工跳楼事件」也让我越发觉得理财的重要性，很庆幸自己刚毕业就有了理财的觉悟，也是以后要持续学习提高的地方。

展望 2018

新的一年给自己定下了几个目标，希望在下一年写总结的时候能一一实现。

  1. 每周运动一次

    刚毕业那段时间会定期去打篮球的，但是天气冷了就不想动了（哎，还是懒），俗话说「身体是革命的本钱」，希望新的一年能把定期健身运动给坚持下来。

  2. 学习前端相关知识

    去年「大前端」就是很火的一个词了，而且近来微信更是推出了小游戏，未来更是有向「大前端」靠近的趋势，新的一年就是要学习前端的相关知识，提升自己的职场竞争力，争取做一个小程序出来。

  3. 总结记录

    新的一年要养成写作记录的习惯。工作后渐渐发现平时工作中遇到沟通不清楚的地方如果能通过把问题写清楚后再说出来，往往问题能迎刃而解了，所以写作能提高人的语言表达能力，而且定期的写作总结能对知识进行梳理，更好地检验自己对知识的掌握程度。

  4. 学习 Python

    去年是人工智能爆发的一年，而人工智能使用最多的就是用 Python 写的，更何况 Python 不止于人工智能，还可以进行数据分析、爬虫、前端、后台。新的一年要入门 Python 并能进行简单的爬虫和数据分析。多学一门语言总归不是坏处，说不定能给自己带来意外的机会呢。

当然，实现这几个目标的前提依然是要做好自己的老本行 Android 开发，不然就本末倒置了。

2018，我来啦！
